API information

  1. Navigate to https://openweathermap.org
  2. Create an account AND verify the account
  3. Go the "My API keys" (https://home.openweathermap.org/api_keys)
  4. Create a new API key
  5. On the "API" page, open the "API doc" for a request (https://openweathermap.org/api)
